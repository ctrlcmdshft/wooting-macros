# Changelog

## v1.1.3-beta — March 2026

### Added
- Always on top option in Application Settings — keeps the window above all other windows

### Fixed
- Sequence toolbar buttons no longer get cut off when the window is resized smaller; delay and clear all are now icon buttons with tooltips
- F7 recording hotkey now works when the Wootomation window is focused
- F7 no longer records itself into the sequence when used as the recording hotkey

### Coming Soon
- Mouse path recording (TinyTask-style) — button is present in the toolbar but disabled while the feature is being finished

### Notes
- We are not responsible for any bans or penalties from using Wootomation in games with anti-cheat. Use at your own risk.

---

## v1.1.2-beta — March 2026

### Added
- Dynamic delay — delay elements now support a random min/max range for more natural timing
- Auto-delay toggle directly in the sequence editor toolbar

### Fixed
- Patch notes incorrectly said "right-click" for the three-dot menu (it's left-click)

---

## v1.1.1-beta — March 2026

### Added
- Toggle macro looping — set a fixed loop count or run until triggered again
- Copy macro to collection via the three-dot menu on macro cards
- Clipboard restore — Paste User Defined String now restores original clipboard content after pasting

### Fixed
- Auto-add delay recording
- Random collection icons
- Context menu z-index
- Various form loading bugs

### Notes
- This build is unsigned. Windows may show a SmartScreen warning on first launch — click "More info" then "Run anyway"

---

## v1.1 — December 2023

### Added
- New UI with animations and updated colors
- Macro search across all collections from the main window
- Macros now work in games (use in multiplayer at your own risk)
- Key combos (Ctrl+Shift+Key etc.) now work correctly

### Fixed
- Security dependency updates
- Any keyboard layout can now be used for a trigger key
